Advantages of React Server Side Components

- Load Faster - They dont have to wait for the Javascript to load
- Smaller client bundle size
- SEO Friendly
- Access to resources that the client cant Access
- Hide sensitive data from the client
- More secure against XSS attacks
- Improved developer experience

Disadvantages

- Not as interactive
- No component state. We cannot use the useState hook
No component lifecycle methods - We cannot use the useEffect hook
